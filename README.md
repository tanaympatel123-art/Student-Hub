# Student Hub

## Student Hub 
**Student Hub** is a website which helps students to be kept updated about events and assignments.Along with that it has faculty, admin and student dashboards for different works and also has things like attendance tracker, event calender and grade book.

## Modules
- Login Page
- Home Page
- Student Dashboard
- Faculty Dashboard
- Admin Dashboard
- Subject List
- Attendance Tracker
- Grade Book
- Event Calender
- Settings

## User Roles
- **Student** can view their grades, can see what events are being conducted when and what is their attendance to be on top of their academic game.
- **Faculty** can upload assignments, put grades and update attendnace of students.
- **Admin** can do things like manage accounts and make department-wide announcements.
